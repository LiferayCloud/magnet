## Setup

1. Make sure you have [node and npm](https://nodejs.org/en/download/) installed:

  ```sh
node -v && npm -v
  ```

2. Install magnet:

  ```sh
npm install magnet
  ```

## Usage

* Build the api, serve it locally, and watch for any changes:

  ```
magnet start
  ```
